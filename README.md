# High-Mountain-Asia-Lake-Water-Level-Dataset-from-CryoSat-2-and-ICESat-2
This interactive map provides a visual interface to explore lake water level time series derived from satellite altimetry (CryoSat‑2 and ICESat‑2) across High Mountain Asia. Open the "Interactive Lake Map.html" in browser. Each colored polygon represents a lake with available observations. click on a polygon, a windowto will popup on the map showing the water level time series, including mission-specific data points, measurement uncertainties, and basic statistics.

The associated timeseries dataset in NetCDF format and lake boundaries can be downloaded from ICIMOD's Regional Database System. 

The associated science paper can be found https://doi.org/10.5194/egusphere-2026-808. 

For data citation please refer to the ICIMOD's Regional Database System. 

For paper citation please refer to the article https://doi.org/10.5194/egusphere-2026-808.
